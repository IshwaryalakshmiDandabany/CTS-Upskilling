Exercise 2.1: Explore Bootstrap directory structure

When you download Bootstrap (or clone it), inspect the folders. This page lists the common purpose of key folders.

  • css/
    Contains compiled stylesheets (e.g., bootstrap.min.css) used by the browser.
  • js/
    Contains Bootstrap JavaScript plugins (e.g., bootstrap.bundle.min.js includes Popper).
  • icons/
    Typically provided by Bootstrap Icons package: vector icon fonts/SVGs for use in your pages.
Tip: Open your local downloaded Bootstrap folder and compare the real folder names/contents to these explanations.
